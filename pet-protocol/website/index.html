// HOW IT WORKS

4 步跑起来

01
克隆仓库

从 GitHub 拿代码，整个 repo 包含 Chrome 扩展 + 本地 bridge。

git clone https://github.com/xiaozihao/pet-protocol
cd pet-protocol
02
启动 pet-bridge

本地 Go 服务，包装 Claude Code / Codex CLI，读取你的记忆文件。

cd bridge && go build -o pet-bridge . && ./pet-bridge
03
加载扩展

Chrome → 扩展程序 → 开发者模式 → 加载已解压的扩展程序，选择 pet-protocol 目录。

chrome://extensions → Load unpacked → /pet-protocol
04
配置 & 享受

点击扩展图标，选择 Agent 模式，设置宠物名字，点 Save。你的 AI 分身就会出现。
