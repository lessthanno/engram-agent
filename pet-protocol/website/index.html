WHY PET PROTOCOL
More than a chat widget.

It reads your memory, knows your page, and talks to teammates' pets.

🧠
Memory-driven personality
Reads your memory files and git log. Knows what you shipped, what you're stuck on, and how long you've been awake.
🌊
True 3D feel
CSS perspective tilt follows your cursor. Sine-wave float animation. Dynamic glow by emotional state. Ground shadow scales with height.
👥
Multi-pet presence
When teammates visit the same page, their ghost cats appear on your screen. 2 cats, 3 cats — no config needed. Just heartbeats.
🔌
Three agent modes
Claude Code via local bridge, Codex via bridge, or direct Anthropic API. Switch in the popup. Pet announces the change.
🎭
Emotional states
idle · celebrate · worried · sleep · hungry · jump. State driven by your memory context. Yellow glow when celebrating, red when worried.
⚡
Zero cloud dependency
The bridge is a single Go binary, no external deps. Your memory stays local. API key never leaves your machine.
TEAM PRESENCE
Your pets meet.

When a teammate opens the same page with Pet Protocol installed, their ghost cat slides in alongside yours. Hue-rotated, semi-transparent, name-tagged. No setup — just heartbeats every 10s to a shared Go bridge.

Coming in v0.5: pets pass speech bubbles. Your message to your pet appears visible to your teammates too.

ROADMAP

v0.1 ✓

Foundation

Pixel cat peeks from right edge. Memory-driven state. Three agent modes. Multi-pet presence server. Bridge in Go stdlib.

Alive & Tangible

3D tilt + float. jimeng AI-generated cat portraits. Dynamic glow. Bug fixes: chat scroll, retreat timer, peek visibility.

v0.3

True 3D

Spline / Three.js model. 5 3D animals. Idle / walk / jump / sleep animations. Mouse parallax via postMessage.

v0.4

Page Awareness

Content script parses current page. Pet reacts to GitHub PRs, Notion docs, Google Docs. Highlight text → instant actions.

v0.5

Team Chat Layer

Pets pass speech bubbles. /whisper @Alice. Broadcast to all teammates. Presence server upgrades to WebSocket.

v0.6

Memory Writing

Pet captures insights. Bridge writes back to memory files. /remember command. Daily synthesis.
